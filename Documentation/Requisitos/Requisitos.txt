	FUNCIONAIS
		RQ_FUNC01 :: Gerar um Formulário de empréstimo (em PDF, após o empréstimo estar concluído)
			Descrição: Um formulário como comprovante de empréstimo, gerado em tempo real pelo programa, onde deverá conter:
				- data de empréstimo;
				- data de devolução;
				- dados do emprestante;
				- Dados do objeto em empréstimo:
					-Quantidade, Nº do Patrimonio, Descrição, Localização,
					 Nº da Nota Fiscal, Data de Aquisição, Fornecedor,
					 Valor R$, Grau de Depreciação.
				- assinatura do responsável;
				- dados da empresa.
			Entrada: Poderá ser solicitado logo após a finalização ou ao acessar as tabelas de empréstimos realizados, com uma opção de 					 gerar formulário.

		REQ_FUNC02 :: Cadastro do tipo de usuário:
			
			- Admin:
				- Íntegra de todos os dados;
				- Todos os Funcionários;
				- criar conta de funcionário.
			- Funcionário:
				- cadastro de emprestante;
				- empréstimo;
				- inserção de item.
		- Sistema necessáriamente precisa ser Windows, devido ao uso de uma biblioteca do sistema, não funcionando no Linux.
		 COM RELAÇÃO A INTERFACE:
			- Visualização das tabelas do banco de dados;
			- Opção de abrir um arquivo (.xls e .csv serão suportados), no qual o sistema fara uma leitura dos dados na planilha e 				  inserí-los no Banco de Dados, lidando com dados incosistentes e duplicados e gerenciando as quantidades dos itens, mantendo 				  uma organização e separação de dados individuais (nota Fiscal, Fornecedor, Nº do Patrimonio, data de aquisição etc.). Tal 				  planilha será armazenada em cópia dentro da pasta "tmp", por motivos de backup e evitar a perda de dados no arquivo 				  original, no qual o sistema irá fazer a leitura por meio desta planilha e não da original;
			- O sistema não fará alterações na planilha aberta no programa, apenas no arquivo DB local;
			- Opção de Salvamento, onde o sistema irá salvar os dados no Banco, mantendo a planilha aberta inalterada;
			- Opção de Gerar um Formulário em PDF (podendo ser impresso), essa opção deve ser mostrada ao realizar um empréstimo e ao 				  acessar o Banco de Dados de Empréstimos, deverá ter a opção de gerar um formulário (será feito em tempo real, não será 				  armazenado no sistema, para diminuir o espaço);
		NÍVEIS DE ACESSO:

			-FUNCIONÁRIO
